# 데이터베이스

### sql,sql개발자의 정의 
sql은 데이터베이스를 직접적으로 엑세스할 수 있는 언어로 
테이블로 만들어져 원하는 데이터를 불러올 수 있다.

필요성: 관계형 데이터베이스, 정보시스템은 sql사용이 필수적인 요소다.

sql 개발자는 데이터베이스와 데이터 모델링에 대한 지식을 바탕으로 응용 소프트웨어를 개발하며, 데이터를 조작하고 최적의 성능을 발휘하는 sql를 작성하는 사람
